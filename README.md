Project Overview
This project compares four machine learning algorithms: Decision Trees, Support Vector Machines, K- Nearest Neighbour and Artificial Neural Networks on a health dataset with risk levels (Low, Medium, High).

The workflow includes:
-Feature engineering (creating new columns like PulsePressure, MAP)
-Feature selection (dropping irrelevant features)
-Train/test split (80–20)
-2‑fold cross‑validation on training data
-Model training and evaluation
-Comparison of classification accuracy and discussion of strengths/weaknesses

 Workflow
1. Data Preprocessing
-Cleaning dataset, handling missing values
-Feature engineering (MAP, PulsePressure, Age groups)
-Feature selection (dropping irrelevant columns)

2. Model Training
-Splitting dataset (80% train, 20% test)
-Training
-Evaluating on test set

3. Evaluation Metrics
-Cross‑validation accuracy
-Test accuracy
-Strengths and weaknesses of each model
